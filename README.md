## 프로젝트 특징
- 따로 데이터베이스를 사용하지 않고 데이터를 .txt 파일로 관리
- .txt 파일에 저장된 데이터를 BufferedReader로 읽어와서 java Collection에 담아서 조작
- 저장해야할 데이터는 BufferedWriter 를 이용해서 .txt파일에 저장

## 후기 
JAVA 팀 프로젝트를 진행하면서 설계의 중요성을 다시 한 번 느꼈습니다.  
최대한 완벽하게 하려고 했던 설계였지만 막상 개발에 들어가고 나니 바뀌어야 되는 부분과 허점이 너무 많았습니다.   
그로 인해 포기해야만 했던 기능들이 있었는데 그 부분이 많이 아쉬웠습니다.   
그리고 이번 팀 프로젝트의 팀장을 맡게 되었는데 팀 프로젝트가 원활히 진행되려면 서로의 진행상황과 의견을 조율하는 것이 가장 중요하다는 점도 느꼈습니다.   
부족한 팀장을 조들이 잘 따라와줘서 프로젝트를 무사히 마무리한 것 같아 조원에게 감사한 마음입니다.
